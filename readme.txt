# YG SHOP - Boutique de Bijoux et Accessoires

## Description
YG SHOP est un site e-commerce de bijoux et accessoires de luxe.
Le site propose une collection pour femme et homme avec des bijoux
de qualité comme Cartier, Van Cleef, Pandora et plus encore.

## Pages du site
- index.html : Page d'accueil avec carousel et catégories
- content/produit.html : Page détail d'un produit
- content/commande.html : Formulaire de commande
- content/inscription.html : Page d'inscription
- content/connexion.html : Page de connexion

## Fonctionnalités
- Affichage dynamique des produits depuis un tableau JavaScript
- Filtrage par catégorie (Femme / Homme)
- Changement de couleur des bijoux
- Formulaire de commande avec validation RegEx
- Inscription et connexion avec localStorage
- Base de données Firebase pour les commandes
- Site responsive

## Instructions d'utilisation
1. Ouvrir index.html dans un navigateur
2. Naviguer dans la boutique
3. Cliquer sur "voir" pour voir le détail d'un produit
4. Cliquer sur "Commander maintenant" pour passer une commande
5. Remplir le formulaire et confirmer

## Base de données
Les commandes sont sauvegardées en temps réel dans Firebase Firestore.
Pour voir les commandes :
1. Aller sur https://console.firebase.google.com
2. Se connecter avec le compte Google du projet
3. Cliquer sur YG SHOP → Firestore Database
4. Voir toutes les commandes dans la collection "commandes"
